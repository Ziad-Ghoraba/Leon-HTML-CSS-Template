# Leon Template One

This is a minimal and creative agency web template using only **HTML** and **CSS**. It showcases essential sections like **Services**, **Portfolio**, **About**, and **Contact**.

## Features

- **Responsive Design:** The template adapts to various screen sizes, ensuring a great user experience on mobile, tablet, and desktop devices.
- **Grid Layout:** The layout is built using a grid system, which helps to maintain consistency and a clean structure.
- **Font Awesome Integration:** Font Awesome icons are used for a variety of features, providing a modern look.
- **Google Fonts:** The template includes custom fonts such as **Work Sans**, **Cairo**, and **Jomhuria** from Google Fonts.

## Sections

- **Header:** Contains a navigation bar with links to different sections (Services, Portfolio, About, Contact).
- **Landing Section:** Welcomes visitors with an introductory message.
- **Services:** Lists the services provided with icons and short descriptions.
- **Portfolio:** Displays project images with brief information.
- **About:** Provides details about the agency with an image and a description.
- **Contact:** Includes an email link and social media icons (YouTube, Facebook, Twitter).

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/leon-template-one.git
    ```
2. Open the `index.html` file in your browser to view the template.
